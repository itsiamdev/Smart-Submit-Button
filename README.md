# Smart Submit Button 🚀

An interactive and playful form validation demo where the submit button cleverly dodges away if the username or password doesn't meet the validation criteria! 🔒 This encourages users to enter valid information in a fun way. 😄

[Live Demo](https://iam269.github.io/Smart-Submit-Button/)

## Features ✨
- **Real-time Validation**: Instant feedback on username and password fields with color-coded borders (red for invalid, green for valid).
- **Username Requirements**: Must start with a letter, contain 4-33 alphanumeric characters.
- **Password Requirements**: At least 8 characters, including uppercase, lowercase, number, and special character.
- **Smart Submit Button**: Moves horizontally when hovered if inputs are invalid, preventing accidental submissions. 🎯

## Technologies Used 🛠️
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Use 📖
1. Open `index.html` in your browser.
2. Enter a username and password.
3. Try hovering over the submit button with invalid inputs – watch it move! 😉
4. Only when both fields are valid can you successfully submit.

Enjoy the interactive experience! 🎉
